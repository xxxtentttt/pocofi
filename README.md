AI-powered DeFi Assistant built for iOPn Builders.

- Portfolio Dashboard
- Smart Swap
- AI Assistant
- Risk Analyzer
- Yield Discovery

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- ethers.js
- OpenAI API
